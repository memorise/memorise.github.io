Android Studio 插件

发表于 2016-04-26   |   分类于 Android Studio   |  

收集一些比较有用或有趣的Android Studio插件

以下插件都可以在Settings->Plugins->Browse repositories搜索到，如果没有会另外给出链接

GsonFormat

将json字符串生成JavaBean.快捷键Alt+s或者Alt+Insert选择GsonFormat

Android Parcelable code generator

JavaBean的序列化，快速实现Parcelable接口.使用：Alt+Insert选择Parcelable

Lifecycle Sorter

根据Activity或者fragment的生命周期对其生命周期方法位置进行先后排序，快捷键Ctrl + alt + K，或者菜单栏选择Code->Sort Lifecycle Methods

findBugs-IDEA

查找bug，避免或预防一些不必要的错误。非常好用，具体使用方法就自已去找吧，GO~

JsonOnlineViewer

调试接口，有get、post等方式可选，返回的数据是json格式才有效，打开方式View->JSONViewer

Android Styler

根据xml生成style代码
使用方法：
首先选择需要转换成style的xml代码

然后找到style.xml，按下快捷键Ctrl+Shift+D或者右键->Paste Style，输入想要生成的style名称

Enter，完成！

folding-plugin

根据资源文件的命名前缀进行分组
使用方法：找到需要分组的资源文件夹，右键->Group

分组就完成了

ECTranslation

英文翻译为中文的插件
使用方法：选中代码，选择 Edit -> Translate，也可以自定义快捷键Preferences -> Keymap -> 搜索Translate - > 右键 add Keyboard Shortcut. 输入你想要的快捷键即可

TranslationPlugin

支持中英文互译的插件，非常好用，下载地址和使用方法在这里
翻译中文：

翻译英文：

LayoutFormatter

格式化XML代码，说到这里，就让我们谈一谈 XML 布局文件代码的优雅性吧

GenerateSerialVersionUID

serializable序列化的时候自动生成SerialVersionUID
使用方法：在实现了Serializable接口的实体类上按Alt+Insert选择SerialVersionUID即可

补充

插件装太多，有一些又不经常使用，Android Studio使用起来卡卡的，怎么办？不加载插件呗o(╯□╰)o，Settings->Plugins-取消勾选不需要加载的插件
